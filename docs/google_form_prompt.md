# Googleフォーム自動作成プロンプト

以下のプロンプトをChatGPTやGeminiなどのAIチャットに貼り付けて実行し、出力されたスクリプトをGoogle Apps Scriptで実行してください。

---

```text
Googleフォームを自動作成するためのGoogle Apps Script (GAS) を書いてください。
このフォームの回答はCSVとしてダウンロードされ、特定のプログラムで読み込まれるため、
質問のタイトル（項目名）は正確に以下の通りに設定してください。

【要件】
フォームタイトル: スプレーマム生育記録
フォームの説明: ハウスごとの植え付けデータを入力してください。

【項目リスト】
1. タイトル: ハウス名
   - 形式: 記述式（Short Answer）
   - 必須: はい

2. タイトル: 品種
   - 形式: 記述式（Short Answer）
   - 必須: はい

3. タイトル: 面積
   - 形式: 記述式（数値）
   - 説明: 坪数を入力（例: 100）
   - 必須: はい
   - バリデーション: 数字のみ

4. タイトル: 消灯日
   - 形式: 日付（Date）
   - 必須: はい

5. タイトル: 花色
   - 形式: プルダウン
   - 選択肢: 白, 黄, ピンク, 赤, オレンジ, 紫, 緑
   - 必須: いいえ（デフォルト処理ありのため）

6. タイトル: 花形
   - 形式: プルダウン
   - 選択肢: シングル, デコラ, アネモネ, スパイダー, ポンポン
   - 必須: いいえ

7. タイトル: 週数
   - 形式: 記述式（数値）
   - 説明: 開花所要週数（空欄なら7週になります）
   - バリデーション: 数字のみ
   - 必須: いいえ

8. タイトル: 係数
   - 形式: 記述式（数値）
   - 説明: 生育係数（空欄なら季節から自動計算）
   - バリデーション: 数字のみ
   - 必須: いいえ
```
